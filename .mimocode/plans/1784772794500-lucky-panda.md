# 首页轮播图 + 分区顺序调整

## 概述
在首页顶部添加可配置的轮播图，并支持在设置中手动调整首页各分区的显示顺序和隐藏。

---

## 一、轮播图功能

### 数据来源
- 轮播图内容来自可配置的分区（视频/音乐/漫画/电子书）
- 视频分区使用 `fanartUrl`（横屏封面），其他分区使用 `coverUrl`
- 默认显示视频分区的轮播，最多 10 项

### UI 实现
- 使用 `HorizontalPager`（已包含在 `foundation` 依赖中）
- 复用现有 `WideMediaCard` 组件（横屏卡片，带渐变遮罩和文字）
- **自动轮播 3 秒间隔**，用户触摸时暂停
- 底部显示圆点指示器
- 无数据时隐藏轮播图区域

### 配置项
- "轮播图来源"：视频/音乐/漫画/电子书（单选）
- "显示轮播图"：开关，关闭后首页不显示轮播

---

## 二、分区顺序 + 隐藏

### 存储（PrefsManager）
- `homeSectionOrder`: JSON 数组 `["videos","music","comics","ebooks"]`
- `homeSectionVisible`: JSON 对象 `{"videos":true,"music":true,"comics":true,"ebooks":true}`
- `carouselSource`: String，默认 `"videos"`
- `isCarouselEnabled`: Boolean，默认 true

### 设置 UI
"首页布局"区域包含：
1. 显示轮播图 — Switch
2. 轮播图来源 — SegmentedButton 单选
3. 分区列表 — 每项：分区名 + 上移/下移按钮 + 显示/隐藏 Switch

## 三、修改文件清单

### 1. `util/PrefsManager.kt`
新增 4 个 key 和对应 property（使用 Gson 序列化 List/Map）

### 2. `res/values/strings.xml` + `res/values-zh/strings.xml`
新增：`home_layout`, `carousel_source`, `carousel_enabled`, `home_section_order_desc`, `move_up`, `move_down`

### 3. `ui/home/HomeViewModel.kt`
- `HomeUiState` 新增：`sectionOrder`, `sectionVisible`, `carouselSource`, `isCarouselEnabled`
- `loadHomeData()` 读取偏好

### 4. `ui/home/HomeScreen.kt`
- 接收新参数
- 顶部条件渲染轮播图（`HorizontalPager` + LaunchedEffect 自动翻页 + 指示器）
- 中部根据 `sectionOrder` + `sectionVisible` 动态渲染各分区
- 隐私过滤同时应用于轮播图数据

### 5. `ui/settings/SettingsScreen.kt`
- 新增"首页布局"section（轮播图开关、来源选择、分区顺序列表）

### 6. `MainActivity.kt`
- 传递新状态到 `HomeScreen` 和 `SettingsScreen`
- 状态变更时写入 PrefsManager

### 7. `ui/theme/Dimens.kt`
新增：`carouselHeight`, `carouselHeightTablet`, `indicatorSize`, `indicatorSpacing`

## 四、验证
1. `./gradlew assembleDebug` 编译通过
2. 首页顶部显示轮播图，3秒自动切换，触摸暂停
3. 设置切换轮播来源 → 首页轮播内容变化
4. 设置关闭轮播图 → 首页无轮播
5. 设置调整分区顺序 → 首页顺序变化
6. 设置隐藏分区 → 首页该分区不显示
7. 隐私模式下成人内容不在轮播图和列表中出现
